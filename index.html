About

This calculator uses Table 1 from the TTB Gauging Manual to correct proof readings based on temperature. Table 1 works with hydrometers that are calibrated at 60 degrees Fahrenheit and can handle most readings between 0 and 200 proof; from 1F to 100F. It is limited at very high and very low temperatures. You can view the source code for this calculator and download a typed version of Table 1 here. This is a hand typed table. Please let me know if you find a mistake.
